# Alura Store Challenge

## Objetivos de proyecto

El objetivo del proyecto es ayudarle al Sr Juan a decidir qué tienda vender, ya que quiere conseguir capital para un nuevo emprendimiento. Por lo anterior, hay que determinar que tienda es la menos eficiente en función de los 
datos de ventas historicos que se tienen.

## Código
El proyecto se desaroola en python, utilizando librerías tales como Pandas, Seaborn y Matplotlib, con las cuales se desarrolla el análisis de datos pertinente en función de los objetivos del proyecto.
